# AGENTS.md

`CLAUDE.md` is this repository's source of truth for agent working rules,
architecture notes, and project context.

Before making changes in this repo, read `CLAUDE.md` and follow it unless a
higher-priority system or developer instruction conflicts with it.

When agent rules, architecture notes, or workflow guidance need to change,
update `CLAUDE.md`. Keep this file as a stable pointer only so different coding
agents do not drift across duplicated instructions.
